## Execução: Instalação do ambiente
1. Criar ambiente:
   no terminal do VSCode (ctrl + ç):
   
   pip install -r requirements.txt


#### Download Data & Setup up files for 100K dataset explorarion
Download the 100K MovieLens dataset from:
https://grouplens.org/datasets/movielens/latest/

Create a folder named data, and inside it create a subfolder called 100k.
Place all the downloaded files from the 100k dataset into this folder and unzip them.

#### Download Data & Setup up files for 33M dataset explorarion
Download the 33M MovieLens dataset from:
https://grouplens.org/datasets/movielens/latest/

Create a folder named data, and inside it create a subfolder called Full33M.
Place all the downloaded files from the 33M dataset into this folder and unzip them.
